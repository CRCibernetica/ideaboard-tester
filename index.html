CRCibernetica Ideaboard Serial Plotter

  1. Plug the Ideaboard into a USB port.
  2. Click Connect to select the serial port.
  3. Click Start to begin plotting the serial output. Data Format
    <tag1>:<value1>,<tag2>:<value2>
    eg. sensor1:541,sensor2:875,sensor3:357
    The name of the tag can be any valid Python variable name.
    Up to six values can be graphed at once.
Sorry, Web Serial is not supported. Use Chrome 89 or later.
